# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Japan Seasons API, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email security reports to: **jun@dokos.dev**

We will respond within 48 hours and work with you to assess and address the issue promptly.

## Security Measures

- CodeQL static analysis (automated via GitHub Actions)
- Dependency vulnerability scanning (Dependabot)
- Secrets scanning (TruffleHog)
- Regular security audits

## Acknowledgments

We appreciate the security community's efforts in helping keep our users safe.
